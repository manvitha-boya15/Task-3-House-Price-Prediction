# Task-3-House-Price-Prediction
House Price Prediction using Linear Regression with Python, Pandas, Scikit-Learn, and Exploratory Data Analysis (EDA).

## 📌 Project Overview

This project predicts house prices using the **Linear Regression** algorithm. The workflow includes data preprocessing, exploratory data analysis (EDA), feature encoding, model training, evaluation, and visualization. The goal is to build a machine learning model that can estimate house prices based on various property features.

---

## 📂 Dataset

- **Dataset:** Housing.csv
- **Records:** 545
- **Features:** 13 input features + 1 target variable (Price)

---

## 🛠️ Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📊 Project Workflow

### 1. Data Loading
- Imported the Housing dataset using Pandas.
- Displayed the first few rows to understand the data.

### 2. Data Preprocessing
- Checked for missing values.
- Encoded categorical variables into numerical values.
- Prepared the dataset for machine learning.

### 3. Exploratory Data Analysis (EDA)
- Analyzed relationships between features.
- Generated a Correlation Heatmap.

### 4. Model Building
- Split the dataset into training and testing sets.
- Trained a Linear Regression model.

### 5. Model Evaluation
Evaluated the model using:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

### 6. Visualization
- Correlation Heatmap
- Actual vs Predicted House Prices

---

## 📈 Results

The Linear Regression model successfully predicted house prices with good accuracy.

### Evaluation Metrics

- Mean Absolute Error (MAE): **979679.69**
- Mean Squared Error (MSE): **1771751116594.04**
- Root Mean Squared Error (RMSE): **1331071.42**
- R² Score: **0.649**
